ЗАГРУЗКА В GITHUB

Репозиторий: https://github.com/miromanof/babrbook

1. Откройте репозиторий и нажмите Add file → Upload files.
2. Перетащите папку tilda-assets целиком.
3. Нажмите Commit changes в ветку main.
4. Убедитесь, что в корне репозитория уже есть Farabee-Black.ttf.
5. Вставьте содержимое файла babrbook_tilda_github_min.html в HTML-блок Tilda.

В коде уже прописаны ссылки вида:
https://cdn.jsdelivr.net/gh/miromanof/babrbook@main/tilda-assets/angara.webp
https://cdn.jsdelivr.net/gh/miromanof/babrbook@main/Farabee-Black.ttf

После загрузки GitHub CDN иногда обновляется не мгновенно. Если файл не открылся сразу, обновите страницу через несколько минут.
